Detailed Description

LandTypeFractions are used to describe 'type of land' like glacier, lake, reservoir and forest.

It is designed as a part of GeoCellData (could be nested, but we take python/swig into consideration). It's of course questionable, since we could have individual models for each type of land, - but current approach is to use a cell-area, and then describe fractional properties.
