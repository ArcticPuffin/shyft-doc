Detailed Description

For temperature inverse distance, also provide default temperature gradient to be used when the gradient can not be computed.

Note
if gradient_by_equation is set true, and number of points >3, the temperature gradient computer will try to use the 4 closes points and determine the 3d gradient including the vertical gradient. (in scenarios with constant gradients(vertical/horizontal), this is accurate)
