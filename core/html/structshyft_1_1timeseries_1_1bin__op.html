Detailed Description

template<class A, class B, class O, class TA>
struct shyft::timeseries::bin_op< A, B, O, TA >

Basic math operators.

Here we take a very direct approach, just create a bin_op object for each bin_op :

  1. specialize for double vs. ts
  2. use perfect forwarding of the arguments, using d_ref to keep shared_ptr vs. object the same
