Detailed Description

template<class S, class TA>
class shyft::timeseries::direct_accessor< S, TA >

The direct_accessor template is a fast accessor.

that utilizes prior knowledge about the source ts, e.g. fixed regualar intervals, where each values represents the average/representative value for the interval. This is typical for our internal timeseries results, and we need to utilize that.

Note
life-time warning: The supplied constructor parameters are const ref.
Template Parameters
Sthe point_source that need to relate to the supplied timeaxis T
TAthe timeaxis that need to match up with the S
See Also
average_accessor that provides transformation of unknown source to the provide timeaxis
