Detailed Description

template<class timeaxis, class temperature_ts, class precipitation_ts, class radiation_ts, class relhum_ts, class windspeed_ts>
struct shyft::core::environment< timeaxis, temperature_ts, precipitation_ts, radiation_ts, relhum_ts, windspeed_ts >

environment supplies the temperature, precipitation,radiation,rel_hum and wind_speed time-series representative for the cell area,height and mid_point Each property can have its own ts-type, so that we can easily represent constants if needed. We use this template class as a building block of the cell-types, since most cells need this information. environment variant with const relative humidity and wind (speed up calc &reduce mem)
