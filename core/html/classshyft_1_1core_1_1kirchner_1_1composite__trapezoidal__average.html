template<class S>
class shyft::core::kirchner::composite_trapezoidal_average< S >

Compute the average of the solution using a n step composite trapezoidal rule.

During the integration of the Kirchner model from to using a dense solver, we can interpolate the solution on the time sub-interval , where is implicitly determined by the tolerance of the ode solver for some internal time step , where . At this sub interval, we use the composite trapezoidal rule to compute the average of each sub-interval using sub-sub-intervals. This is a costly routine, requiring interpolations at each sub-interval.
