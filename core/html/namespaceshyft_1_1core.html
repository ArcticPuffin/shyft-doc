struct  cell
 cell template for distributed cell calculations. a cell have :

  1. geo information (.mid_point(),area(), landtype fractions etc..)
  2. parameters reference to parameters for the cell method-stack (subject to calibration)
  3. state the current state of the cell, like water-content etc.
  4. env_ts provision the environmental properties (prec,temp,rad,wind,rhum) at the cell.mid_point
  5. sc state collector, plays its role during run(), collects wanted state each step
  6. rc response collector,plays its role during run(), collects wanted response each step
