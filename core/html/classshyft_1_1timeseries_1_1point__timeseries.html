Detailed Description

template<typename TA>
class shyft::timeseries::point_timeseries< TA >

A point source with time_axis.

The point_source template class contains a time axis and an equally sized vector of corresponding values. It implements the shyft::timeseries::point_source_with_no_regularity interface and the Accessor interface, in addition to a custom interface for efficient time series arithmetics.

Template Parameters
TATime axis type that supports:
  1. .index_of(tx) const –> -1(npos) or lowerbound index
  2. .size() const –> number of periods on time-axis
  3. op() (i) const –> utcperiod of the i'th interval
