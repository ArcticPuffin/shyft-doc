Detailed Description

template<class S>
class shyft::core::kirchner::trapezoidal_average< S >

Compute the average of the solution using a simple trapezoidal rule.

During the integration of the Kirchner model from t_0 to t_1, this routine uses the intermediate solution steps at t_i where t_0 < t_i <= t_1, implicitly given by the tolerance of the ode solver, to apply a simple trapezoidal rule to compute the average. This routine is not very costly, as it only uses the already computed intermediate solution steps during the calculations.
