Detailed Description

Specialized kalman filter for temperature (e.g.:solar-driven bias patterns)

The observation point (t,v) is folded on to corresponding period of the day (number of periods in a day is parameterized, typically 8). A simplified kalman filter algorithm using the forecast bias as the state-variable. Observed bias (fc-obs) is feed into the filter and establishes the kalman best predicted estimates (x) for the bias. This bias can then be used as a correction to forecast in the future to compensate for systematic forecast errors.

Credits: Thanks to met.no for providing the original source for this algorithm.
