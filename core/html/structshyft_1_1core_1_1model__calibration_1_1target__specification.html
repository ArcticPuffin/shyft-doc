Detailed Description

template<class PS>
struct shyft::core::model_calibration::target_specification< PS >

The target specification contains:

  1. a target ts (the observed quantity)
  2. a list of catchment ids (zero-based), that denotes the catchment.discharges that should equal the target ts.
  3. a scale_factor that is used to construct the final goal function astemplate helper classes to be used in enable_if_t in the optimizer for snow swe/sca:

goal_function = sum of all: scale_factor*(1 - nash-sutcliff factor) or KLING-GUPTA

Template Parameters
PStype of the target time-series, any type that is time-series compatible will work. Usually a point-based series.

Parameters
tsthe target time-series that contain the target/observed discharge values
cidsa vector of the catchment ids in the model that together should add up to the target time-series
scale_factorthe weight that this target_specification should have relative to the possible other target_specs.
calc_modehow to calculate goal-function nash-sutcliffe or the more advanced and flexible kling-gupta
s_rscale factor r in kling-gupta
s_ascale factor a in kling-gupta
s_bscale factor b in kling-gupta
catchment_property_determines if this target specification is for DISCHARGE|SNOW_COVERED_AREA|SNOW_WATER_EQUIVALENT
uida user supplied uid, string, to help user correlate this target to external data-sources
