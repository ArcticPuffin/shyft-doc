Detailed Description

Simple parameter struct for the hbv_stack method stack.

This struct contains the parameters to the methods used in the HBV assembly.

Template Parameters
PTParameterPriestleyTaylor parameter type that implements the interface:
  • PTParameter.albedo const –> double, land albedo parameter in PriestleyTaylor.
  • PTParameter.alpha const –> double, alpha parameter in PriestleyTaylor.
SnowStateHBVSnow parameter type that implements the parameter interface for HBVSnow.
KStatehbv_tank parameter type that implements the parameter interface for hbv_tank.
