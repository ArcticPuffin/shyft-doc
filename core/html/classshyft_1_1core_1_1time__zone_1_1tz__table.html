Detailed Description

The tz_table is a table driven approach where each year in a specified range have a dst information containing start/end, and the applied dst offset. This approach allows to have historical correct dst-rules, at minor space/time overhead E.g. Norway, summertime rules are changed to EU defs. in 1996 first time applied was 1916, then 1943-1945, then 1959-1965, then 1980-to 1996, where eu rules was introduced.
