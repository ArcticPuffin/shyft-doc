Detailed Description

Yet another variant of calendar time-axis.

This one is similar to calendar_dt, except, each main-period given by calendar_dt have sub-periods. E.g. you would like to have a weekly time-axis that represents all working-hours etc. If the sub-period(s) specified are null, then this time_axis is equal to calendar_dt.

Note
this is a non-continuous| sparse time-axis concept
