Detailed Description

a generic (not sparse) time interval time-axis.

This is a static dispatch generic time-axis for all dense time-axis. It's merely utilizing the three other types to do the implementation. It's useful when combining time-axis, and we would like to keep the internal rep. to the most efficient as determined at runtime.
