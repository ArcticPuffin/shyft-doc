Detailed Description

template<typename TA, typename F>
class shyft::timeseries::function_timeseries< TA, F >

A point source with a time_axis, and a function Fx(utctime t) that gives the value at utctime t, where t is from timeaxis(i).t suitable for functional type of time series.

Template Parameters
TATime axis type that supports:
  1. .index_of(tx) const –> -1(npos) or lowerbound index
  2. .size() const –> number of periods on time-axis
  3. op() (i) const –> utcperiod of the i'th interval
Ffunctor object that supports
  1. op()(utctime t) const –> double, F(t)
