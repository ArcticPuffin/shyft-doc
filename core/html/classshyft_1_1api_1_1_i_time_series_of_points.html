Detailed Description

class ITimeSeriesOfPoints: is a simple Native timeseries class interface used to marshal timeseries data between the native client libraries and python wrapper.

In the c++ core, timeseries are templates, - but in the python parts we need concrete classes that we can rely on (stable/efficient interfaces).

It should reflect the interfaces/concept of the core timeseries sufficient to put data into the core-model, and to extract data back.
