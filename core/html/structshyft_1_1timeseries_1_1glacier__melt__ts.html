template<class TS_A, class TS_B = TS_A>
struct shyft::timeseries::glacier_melt_ts< TS_A, TS_B >

glacier melt ts

Using supplied temperature and snow covered area[m2] time-series computes the glacier melt in units of [m3/s] using the the following supplied parameters:

  1. dtf:day temperature factor (dtf),
  2. glacier_area_m2: glacier area in [m2] units
Template Parameters
TSa time-series type
Note
that both temperature and snow covered area (sca) TS is of same type shyft::core::glacier_melt::step function
