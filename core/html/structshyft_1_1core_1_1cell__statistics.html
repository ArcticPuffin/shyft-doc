Detailed Description

cell statistics provides ts feature summation over cells

Since the cells are different, like different features, based on method stack, we use template to do the "main work", either to do a scaled average, e.g. temperature, or plain sum (cell run_off in m3/s) Using this technique, there is only one line pr. feature sum/average needed -and we can keep the kloc down to a minimum.

Using swig/python, we need a helper class in either _api.h, or api.i to provide the functions to python layer
