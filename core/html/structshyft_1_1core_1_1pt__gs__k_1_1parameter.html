Detailed Description

Simple parameter struct for the PTGSK method stack.

This struct contains the parameters to the methods used in the PTGSK assembly.

Template Parameters
PTParameterPriestleyTaylor parameter type that implements the interface:
  • PTParameter.albedo const –> double, land albedo parameter in PriestleyTaylor.
  • PTParameter.alpha const –> double, alpha parameter in PriestleyTaylor.
GSStateGammaSnow parameter type that implements the parameter interface for GammaSnow.
KStateKirchner parameter type that implements the parameter interface for Kirchner.
