Detailed Description

template<class S, class TA>
class shyft::timeseries::accumulate_accessor< S, TA >

provides the accumulated value accessor over the supplied time-axis

Given sequential access, this accessor tries to be smart using previous accumulated value plus the new delta to be efficient computing the accumulated series from another kind of point source.
