Detailed Description

template<class C>
class shyft::experimental::repository::cell_file_repository< C >

a simple cell_repository, responsible for providing cells of supplied template parameter type C with geo_cell_data from somewhere, 'somewhere' is the files with specific names for the features:

  1. forest
  2. glacier
  3. lake (and reservoirs)
  4. reservoir mid-points (to figure out the reservoirs from the lakes)
  5. catchment map
  6. a digital terrain model (dtm) in a sub- directory . These are read into memory shyft::experimental::io::slurp and processed by shyft::experimental::wkt_reader into boost::geometry multipolygons then feed into the geo_cell_data_computer that uses the rudimentary region_grid class to provide a grid-type of geometry, 1km x 1km squares.
Template Parameters
Ccell type that supports construction, state_t and parameter_t
