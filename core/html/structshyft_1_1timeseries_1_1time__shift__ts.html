Detailed Description

template<class Ts>
struct shyft::timeseries::time_shift_ts< Ts >

time_shift ts do a time-shift dt on the supplied ts

The values are exactly the same as the supplied ts argument to the constructor but the time-axis is shifted utctimespan dt to the left. e.g.: t_new = t_original + dt 

  lets say you have a time-series 'a'  with time-axis covering 2015
  and you want to time-shift so that you have a time- series 'b'data for 2016,
  then you could do this to get 'b':

      utc = calendar() // utc calendar
      dt  = utc.time(2016,1,1) - utc.time(2015,1,1)
       b  = timeshift_ts(a, dt)
